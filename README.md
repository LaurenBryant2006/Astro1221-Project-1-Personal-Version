**Astro 1221 Project 1 - Custom Constellation Generator**

For this project, my partner and I created code to generate random star maps and corresponding constellations, along with a separate piece of code to create a Streamlit pop-up.
Initially, before the Streamlit idea, we animated the sky map, so you will see remnants of the animation code we had to freeze to make it work in Streamlit.
I primarily worked on the code for the star maps, while my partner focused on the Streamlit code. 
The file CodeForStreamlit1-repo was mostly my work, and the file Random Constellation App Generator was her work.

It may be a little rough around the edges, sometimes it would work and sometimes it wouldn't. 
This was by far our best attempt, it's certainly not perfect but I am proud of it. 
I hope to come back to this project in the future after I have learned more to see if I can play with it in another way and fix it up.
